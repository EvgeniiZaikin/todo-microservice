This is test ToDo microservice basic on node.js.

Stack:
    - node.js
    - express.js
    - sqlite (sqlite3.js)
    - body-parser.js
    - joi.js
    - jsdoc.js

Start:
    - clone repository with project
    - run  'npm install'
    - run 'npm run start'
    - you can send request with something like postman on 'localhost:3032'
